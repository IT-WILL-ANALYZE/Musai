아래 텍스트는 하나의 문서에서 추출된 원문 내용입니다.

작업 지시:
1. 이 텍스트 안에 "행(row)과 열(column) 구조를 가진 표"가 있다면,
   - 컬럼 이름을 추정하고
   - 각 행을 하나의 JSON 객체로 만들어
   - JSON 배열 형태로 출력하세요.

2. 표가 전혀 없다면,
   - 반드시 "NO_TABLE" 이라고만 출력하세요.

출력 규칙 (매우 중요):
- 반드시 JSON 형식만 출력
- 설명 문장, 마크다운, 주석 금지
- JSON key는 문자열
- 값이 없으면 빈 문자열("") 사용
- 여러 표가 있더라도 하나의 배열로 통합

출력 예시 (표인 경우):
[
  {{
    "컬럼1": "값1",
    "컬럼2": "값2"
  }},
  {{
    "컬럼1": "값3",
    "컬럼2": "값4"
  }}
]

표가 없는 경우:
NO_TABLE

문서 텍스트:
{docs_text}
