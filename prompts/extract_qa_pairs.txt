아래 텍스트를 질문과 답변(Question–Answer) 쌍으로 구조화하세요.

규칙:
- 질문(question)은 반드시 '?'로 끝나거나, 의미상 질문인 문장
- 답변(answer)은 해당 질문에 대한 직접적인 설명 또는 해설
- 질문과 답변은 의미 단위로 묶기
- 답변이 명확하지 않은 경우 빈 문자열("") 사용
- 중복되거나 의미가 같은 질문은 하나로 병합
- 반드시 JSON 배열만 출력 (설명, 마크다운, 주석 금지)

출력 형식:
[
  {{
    "question": "...",
    "answer": "..."
  }}
]

텍스트:
{docs_text}
