아래 텍스트는 PDF 한 페이지(또는 여러 페이지)에서 추출한 원문 텍스트입니다.

작업 지시:
1. 이 텍스트가 "행(row)과 열(column)로 구성된 표"라고 판단되면,
   - 컬럼 이름을 추정하고
   - 각 행을 하나의 객체로 하여
   - JSON 배열 형태로 출력하세요.

2. 표가 아니라면,
   - 반드시 "NO_TABLE" 이라고만 출력하세요.

출력 규칙 (매우 중요):
- 반드시 JSON 형식만 출력
- 설명 문장, 마크다운, 주석 금지
- JSON 키는 문자열로
- 값이 없으면 빈 문자열("") 사용
- 표가 여러 개여도 하나의 배열로 통합

예시 출력 (표인 경우):
[
  {{
    "번호": "1",
    "질문": "가장 만족스러운 점?",
    "분석방법": "긍정 리뷰 키워드 빈도 분석"
  }}
]

표가 아닌 경우:
NO_TABLE

텍스트:
{page_text}
